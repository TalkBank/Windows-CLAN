
This version, the University of Reading version of vocd, includes:

- +g option for computing Limiting Type Type Ratio

- -m option for segmenting words into their component morphemes.

The following additional options that can be used to override the
defaults for curve fitting are commented out by default:

- interactive setting of parameters for computing d_compute
   printf("+Df<n>: D_optimum - size of starting sample (default  35) \n");
   printf("+Dt<n>: D_optimum - size of largest sample  (default  50) \n");
   printf("+Di<n>: D_optimum - size of increments      (default   1) \n");
   printf("+Ds<n>: D_optimum - the number of samples   (default 100) \n");
   printf("-R    : D_optimum - use random sampling with replacement.\n");
   printf("                  - default: random sampling without replacement.\n");
   printf("-E    : D_optimum - use sequential sampling.\n");
